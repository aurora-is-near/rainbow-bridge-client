`@near-eth/utils`
=================
Common utils for connector packages

Installation
------------

```shell
yarn add @near-eth/utils
```

```json
"dependencies": {
  "@near-eth/utils": "^1.0.0",
}
```

Usage
-----
```js
import {
  urlParams,
  lastBlockNumber,
  borshifyOutcomeProof
} from '@near-eth/utils'
```
